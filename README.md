# auto-replay-logging
Rude Telegram auto-reply bot (aiogram 3.x)
- Greets with "привет чувак"
- Ask "как дела" answer "нормас, ты че?"
- Everything else: "я получил твоё сообщение"
 # MAIN FEATURES
- Instant admin notification on new users (User ID + @username + name sent straight to your personal chat)
- Full message logging → bot_messages.log
- Logs new users to new_users.log
- Built with Python + aiogram 3.x , Fully asynchronous, fast, reliable
#IMPORTANT
In the file configq.py you must replace the example token with your own real bot token from @BotFather.
